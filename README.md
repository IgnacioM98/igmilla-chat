## expo run app (Build folders created android/ios)

  

```bash

npx  expo  prebuild  -p  android

npx  expo  prebuild  -p  ios  //  only  macOS

npx  expo  run:android

npx  expo  run:ios  //  only  macOS

```

  

## expo start (No build folders created)

  

```bash

npx  expo  start

```
